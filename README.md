# Hotel-Reservation-System-JavaFX
A complete desktop application built with Java and JavaFX, integrated with an Oracle Database to manage hotel room bookings and employee access .
### How it Works
Employee create account with email and password and can after login add , delete , see and update User booking.
### Features
1. **Authentication System:** Secure Sign-in, Sign-up, and Password Reset functionalities for employees.
2. **Booking Management:** Employees can Add, Update, Select, and Delete guest reservations.
3. **Data Display:** Real-time data rendering using JavaFX `TableView` to display guest details (ID, Name, Room Number, Number of Nights).
### Database
in the application i use a relational database (Oracle) to Hotel Reservation System by
* Create an Oracle database.
* Execute the SQL queries provided in the (DB SQL code.txt) file to create the necessary tables.
* **Language used** :- Java (OOP Principles)
* **Framework** :- JavaFX (UI , Layout)
* **Styling** :- CSS stylesheet (CSS.css)
